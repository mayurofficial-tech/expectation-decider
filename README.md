# ЁЯУК Expectation Decider тАФ Probability & Statistics Project

## ЁЯУМ Project Overview

Expectation Decider is a Probability and Statistics based data analysis project developed using Python and Pandas.

The project analyzes student performance data and predicts the probability of passing a competitive mathematics exam based on:

- Study Hours
- Attendance
- Group Discussion Participation
- Previous Test Scores

This project demonstrates concepts of:

- Probability
- Conditional Probability
- Bayes Theorem
- Random Variables
- Probability Distribution
- Contingency Tables
- Statistical Analysis

---

# ЁЯУБ Dataset Information

The dataset contains information for 200 students.

| Column Name | Description |
|---|---|
| study_hours | Weekly study hours |
| attendance | Attendance percentage |
| group_discussion | Participation in discussions (Yes/No) |
| previous_test_score | Previous test marks |
| final_exam_pass | Final result (Pass/Fail) |
